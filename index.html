Repositório de Transição: VBA para LibreOffice Basic

Guia prático e tabelas de equivalência para conversão de macros e automação de planilhas.

Bem-vindo ao Repositório

Este portal foi idealizado para centralizar a documentação técnica necessária para migrar rotinas corporativas e acadêmicas desenvolvidas em VBA (MS Excel) para o StarBasic (LibreOffice Calc).

Navegue pelas abas acima para encontrar mapeamentos diretos de comandos, entender o comportamento da flag de compatibilidade ou aprender a reescrever seus códigos utilizando a API UNO nativa do LibreOffice.

Comandos Equivalentes Diretos

Comandos básicos de manipulação de dados, caixas de diálogo e funções internas que possuem contrapartidas diretas.

Categoria Comando VBA (Excel) Comando Basic (LibreOffice) Função
Interface MsgBox "Texto" MsgBox "Texto" Exibe uma caixa de mensagem na tela.
Strings InStr(1, str, "busca") InStr(1, str, "busca") Encontra a posição de uma substring.
Vetores UBound(meuArray) UBound(meuArray) Retorna o maior índice disponível em um array.
Tipos CInt(valor) CInt(valor) Converte uma expressão para o tipo Inteiro.

O Comando Option VBASupport 1

Atenção: Esta flag deve ser inserida estritamente na primeira linha do seu módulo de código no LibreOffice para ativar o motor de compatibilidade parcial.

Quando ativo, o LibreOffice Basic consegue interpretar diretamente diversos objetos nativos do Excel sem a necessidade de reescrever a arquitetura UNO.

Objeto / Propriedade VBA Status no LibreOffice Notas de Uso
Worksheets("Plan1").Select Suportado Funciona perfeitamente para seleção de abas ativas.
Range("A1:B10").Value Suportado Leitura e escrita simples de dados estruturados.
ActiveCell.Offset(1, 0) Parcial Pode falhar se o foco da janela principal mudar abruptamente.
Eventos de Planilha (UserForms complexos) Não Suportado Formulários e eventos avançados devem ser reconstruídos do zero.

Alternativas Nativas (API UNO)

Abaixo estão exemplos estruturais de como converter comandos comuns do Excel para o padrão nativo do LibreOffice (sem a utilização da flag VBASupport). Esta é a forma mais performática e estável a longo prazo.

1. Alterar Cor de Fundo de uma Célula

VBA Excel 
Sub AlterarCorVBA()
    Range("A1").Interior.Color = RGB(255, 0, 0)
End Sub
LibreOffice Basic (Nativo) 
Sub AlterarCorBasic()
    Dim oDoc As Object, oSheet As Object, oCell As Object
    oDoc = ThisComponent
    oSheet = oDoc.Sheets(0) 'Primeira planilha
    oCell = oSheet.getCellRangeByName("A1")
    oCell.CellBackColor = RGB(255, 0, 0)
End Sub

2. Copiar e Colar um Intervalo

VBA Excel 
Sub CopiarIntervaloVBA()
    Range("A1:A10").Copy Destination:=Range("B1")
End Sub
LibreOffice Basic (Nativo) 
Sub CopiarIntervaloBasic()
    Dim oDoc As Object, oSheet As Object
    Dim oOrigem As New com.sun.star.table.CellRangeAddress
    Dim oDestino As New com.sun.star.table.CellAddress
    
    oDoc = ThisComponent
    oSheet = oDoc.Sheets(0)
    
    ' Definir intervalo de origem (A1:A10)
    oOrigem.Sheet = 0 : oOrigem.StartColumn = 0 : oOrigem.StartRow = 0
    oOrigem.EndColumn = 0   : oOrigem.EndRow = 9
    
    ' Definir destino (B1)
    oDestino.Sheet = 0 : oDestino.Column = 1 : oDestino.Row = 0
    
    oSheet.copyRange(oDestino, oOrigem)
End Sub
